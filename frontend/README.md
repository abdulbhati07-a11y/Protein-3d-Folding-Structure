# Protein Folding Explorer — Frontend

React 19 + Vite frontend for the Protein Folding Explorer.

## Setup

```bash
npm install
cp .env.example .env.local   # then fill in your Supabase credentials
npm run dev                  # → http://localhost:5173
```

Requires the backend running on `http://localhost:5000` (Vite proxies `/api/*` automatically).

## Build

```bash
npm run build   # output → dist/
```

See the root [README](../README.md) for full project documentation.
